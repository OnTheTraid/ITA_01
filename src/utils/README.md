# utils
Описание 

Файл: utils/logging_stub.py
Заглушка на случай отсутствия терминала при тестах.
Этот логгер будет использоваться всеми подмодулями CoreData до подключения глобального оркестратора логов.
---
Файл: utils/config_loader.py
Загружает параметры .env и config.yaml и объединяет их в единый конфиг-объект.
---
Файл: utils/mt5_stub.py
Заглушка для логов. Используется, если библиотека MetaTrader5 недоступна или терминал не запущен.
Позже он будет автоматически подменяться настоящим API, если MetaTrader5 установлен.
